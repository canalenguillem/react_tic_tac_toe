# Proyecto tic-tac-toe en react

## Instalación de Yarn

npm install -g yarn

## Creación del Proyecto con Vite:

yarn create vite tic-tac-toe --template react-ts
cd tic-tac-toe

## Instalación de Dependencias:

yarn
